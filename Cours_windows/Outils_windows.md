# Outils d'administration Windows


__RSAT__ Outils d’administration de serveur distant (Remote Server Administration Tools) : permet d'administrer un serveur depuis un poste client  
__Gestionnaire de serveur__ pour y accéder une fois installé  

__Windows Admin Center__ (WAC) :  Permet de monitorer toute son infra windows, chaque serveur, ordinateur, lancer du powershell, faire du RDP, lancer des maj, reboot...
