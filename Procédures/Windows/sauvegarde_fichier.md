# Sauvegarde d'un seul fichier automatiquement W11


- Panneau de configuration
- Système et sécurité > enregistrer des copies de sauvegarde de vos fichiers à l'aide de l'Historique des fichiers
- Sauvegarde d'image système
- Configurer la sauvegarde
- Enregistrer sur un réseau...
- Indiquer l'emplacement réseau (ex: \\172.15.44.3\d) 
- Indiquer un compte qui a les droits NTFS + les autorisations sur le partage (controle totale)
- Me laisser choisir (dans le cadre où on souhaite sauvegarder qu'un seul repo)
- Choisir ce qu'on souhaite backup
- Suivre l'assistant..
