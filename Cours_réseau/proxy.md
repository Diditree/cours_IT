# Proxy

__Proxy__ : un proxy est un intermediaire entre des clients et internet  
__Squid__ est un proxy HTTP/S qu'on peut installer sur Linux  

A quoi sert un __proxy__: 
- Il peut stocker en cache : si 50 users vont sur le même site, 1 seule requet sort sur internet
- Il permet de filtrer l'accès à internet, bloquer/autoriser des sites, catégories, par horaires
